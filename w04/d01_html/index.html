W04D01, HTML

Objectives

  • Create semantically-sectioned web content without regard to style
  • Explain the different roles of HTML, CSS, and Javascript, and how they interact with the DOM
  • Explain the difference between tags, self-closing tags, attributes, comments, and DOCTYPE declarations
  • Explain the concept of a "markup language" and how it differs from a programming language
  • Incorporate special characters
  • Validate HTML using online tools
